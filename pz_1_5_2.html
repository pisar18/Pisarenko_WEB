МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Для вказівки типу поточного документа <!DOCTYPE> Це необхідно, щоб браузер розумів, як слід інтерпретувати поточну веб-сторінку,
оскільки HTML існує в декількох версіях.		 HTML
Для каркасу документа <html> Початок і закінчення HTML-документа HTML5
<head> Призначений для зберігання інших елементів,
ціль яких - допомогти браузеру в роботі з даними		 HTML5
Для роботи з текстом <p> Визначає текстовий документ HTML5
<b> Встановлює жирне накреслення шрифту HTML5
<i> Встановлює курсивне накреслення шрифту HTML5
<u> Додає підкреслення до тексту HTML5
Для гіперпосилань <a> Призначений для створення посилань HTML5
Для створення списків <ul> Призначений для створення неномерованих списків HTML5
<ol> Призначений для створення номерованих списків HTML5
Для створення горизонтальної лінії <hr> Одиночний тег, використовується щоб розділити
декілька різних по смислу текстів		 HTML5
Для створення таблиці <table> Це тег, який говорить браузеру, що всередині нього буде таблиця HTML5
Для створення рядка в таблиці <tr> Цей тег позначає рядок HTML5
Для створення комірки в таблиці <td> Цей тег позначає комірку HTML5
Встановлює новий рядок <br> Цей тег встановлює новий рядок в тому місці, де цей тег знаходиться. HTML5
Для виділення фрагмента документа <div> Є блоковим елементом і призначений для виділення фрагмента
документа з метою зміни виду вмісту.		 HTML5
Заголовки <h1>...<h6> Відносяться до блокових елементів, вони завжди починаються з нового рядка,
а після них інші елементи відображаються на наступному рядку.		 HTML
Для відображення на веб-сторінці зображень <img> Призначений для відображення на веб-сторінці зображень в графічному
форматі GIF, JPEG або PNG. Адреса файлу з картинкою задається через атрибут src.		 HTML
